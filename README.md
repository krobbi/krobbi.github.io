# About
krobbi.github.io is a GitHub Pages website hosted at https://krobbi.github.io.
The site contains:
* A homepage.
* A 404 page.
* A page listing software license texts.
* Software license texts hosted in plaintext.
* CSS stylesheets.
* Favicon images.

# Generating
The site is fully static, meaning all of its pages are stored on the server
with their contents fully rendered. This improves page load performance and
compatibility at the expense of storing larger files with duplicate data.

The site's content is generated by running the [Python](https://www.python.org)
app in `generator/` from the root of the repository. The app requires the
packages listed in `requirements.txt` installed to run correctly. The site's
content is output to the `out/` directory.

The Python app is not yet fully implemented.
